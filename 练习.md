## 第六章课后

用程序打印以下图形

     A
    ABA
   ABCBA
  ABCDCBA
 ABCDEDCBA

代码：
```
#include <stdio.h>
int main(){
char ch;
scanf("%c",&ch);
for(char i='A';i<=ch;i++){
       for(int k=(ch-i);k>=0;k--)
                printf(" ");
        for(char j='A';j<=i;j++)
                printf("%c",j);
        for(int h=i-1;h>='A';h--)
                printf("%c",h);
        printf("\n");
}
}
```
